# Node TypeScript Fastify API

## For development purposes only!

Example high-performance backend API built with:

- Node.js
- TypeScript
- Fastify
- Prisma ORM
- SQLite
- Docker

## Features

- REST API
- typed backend
- database ORM
- Swagger documentation
- Docker setup

## Run locally

Install dependencies:
`npm install`

Run migrations:
`npx prisma migrate dev`

Start dev server:
`npm run dev`

API available at:
`http://localhost:3000/articles`

Swagger docs:
`http://localhost:3000/docs`
